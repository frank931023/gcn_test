二分圖共現矩陣 (Bipartite Co-occurrence) 檢測女巫攻擊理論根據： 詐騙農場通常會利用同批設備或 IP 註冊大量人頭戶（女巫攻擊 Sybil Attack）。透過二分圖 (Bipartite Graph) 投影，可以找出隱藏的關聯群體。提取欄位：user_info 的 user_id twd_transfer, crypto_transfer, usdt_twd_trading 記錄中的 source_ip 特徵計算： 建立 User 與 IP 的關聯對應。提取特徵為「該用戶使用過的 IP，同時被多少其他獨立 user_id 使用過」。視覺化方式： 鄰接矩陣熱力圖 (Adjacency Matrix Heatmap)。X 軸為 IP 雜湊值，Y 軸為 User ID，有交集則填色。若圖面上出現方塊狀的密集填色區（Block-diagonal），就代表這群看似無關的 KYC 帳戶，實際上是由同一個犯罪集團在背後操控。
