# U 型迴轉與零餘額脈衝特徵分析結果

6. **balance_step_normal_1~3.png** - 正常用戶餘額階梯圖

## 🎯 關鍵發現

### 最有效的特徵（黑名單用戶顯著更高）：

1. **脈衝次數**: +245.80% ⭐⭐⭐⭐⭐
   - 黑名單用戶: 0.18
   - 正常用戶: 0.05
   - **最具區分度的特徵！**

2. **最大餘額**: +157.86% ⭐⭐⭐⭐
   - 黑名單用戶: 55,458 TWD
   - 正常用戶: 21,507 TWD
   - 人頭戶處理大額交易

3. **U 型迴轉比率**: +129.73% ⭐⭐⭐⭐
   - 黑名單用戶: 0.19
   - 正常用戶: 0.08
   - 大額交易後快速歸零

## 💡 使用建議

### 快速查看順序：

1. 先看 **feature_radar.png** - 整體了解
2. 再看 **feature_barplot.png** - 具體數值
3. 然後看 **balance_comparison.png** - 直觀理解行為差異
4. 最後看個別案例 **balance_step_*.png** - 深入分析

### 模型應用：

- 將「脈衝次數」和「U 型迴轉比率」作為主要特徵
- 結合「最大餘額」進行綜合判斷
- 可以整合到 GCN 或 LightGBM 模型中
